T1.循环移位后的矩阵相似检查

暴力模拟，题目让我移我就移，注意下标的计算

class Solution {
    public boolean areSimilar(int[][] mat, int k) {
        int m = mat.length;
        int n = mat[0].length;
        int[][] help = new int[m][n];
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                int dif = k % n;
                if (i % 2 == 0) {
                    help[i][j] = mat[i][(n - dif + j) % n];
                } else {
                    help[i][j] = mat[i][(j + dif) % n];
                }
            }
        }
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (mat[i][j] != help[i][j]) {
                    return false;
                }
            }
        }
        
        return true;
    }
}

数据范围较小，也可以暴力，不过可以优化为元音字母 +1，辅音字母 -1，就不用申请额外数组了

        char[] sc = s.toCharArray();

        int ans = 0;
        int[] help = new int[2];

                    help[0]++;
                } else {

T3.交换得到字典序最小的数组

利用了分组重排的思想，这道题的核心是将可以互相交换的分为一组，并按最小值依次填空

class Solution {
    public int[] lexicographicallySmallestArray(int[] nums, int limit) {
        int n = nums.length;
        int[][] arr = new int[n][2];
        for (int i = 0; i < n; i++) {
            arr[i][0] = nums[i];
            arr[i][1] = i;
        }
        Arrays.sort(arr, Comparator.comparingInt(x -> x[0]));

        int[] res = new int[n];

        int j = 0;
        while (j < n) {
            List<Integer> index = new ArrayList<>();
            List<Integer> values = new ArrayList<>();

            values.add(arr[j][0]);
            index.add(arr[j][1]);
            int t = j + 1;
            while (t < n && arr[t][0] - arr[t - 1][0] <= limit) {
                values.add(arr[t][0]);
                index.add(arr[t][1]);
                t++;
            }
            
            Collections.sort(values);
            Collections.sort(index);
            for (int s = 0; s < values.size(); s++) {
                res[index.get(s)] = values.get(s);
            }
            j = t;
        }

        return res;
    }
}
T4.统计美丽子字符串 II

赛后训练：

  • 560. 和为 K 的子数组
  • 974. 和可被 K 整除的子数组
  • 1590. 使数组和能被 P 整除
  • 523. 连续的子数组和
  • 525. 连续数组
class Solution {
    public long beautifulSubstrings(String s, int k) {
        char[] str = s.toCharArray();
        int n = str.length;
        int offset = 0;
        for (int i = 1; i <= k; i++) {
            if ((i * i) % k == 0) {
                offset = i * 2;
                break;
            }
        }

        long res = 0;
        HashMap<Integer, HashMap<Integer, Integer>> map = new HashMap<>();
        map.put(0, new HashMap<>());
        map.get(0).put(0, 1);
        int acc = 0;
        for (int i = 0; i < n; i++) {
            if ("aeiou".indexOf(str[i]) != -1) {
                acc++;
            } else {
                acc--;
            }

            int key = (i + 1) % offset;
            if (map.containsKey(acc)) {
                HashMap<Integer, Integer> hashMap = map.get(acc);
                res += hashMap.getOrDefault(key, 0);
            }
            map.computeIfAbsent(acc, x -> new HashMap<>()).merge(key, 1, Integer::sum);
        }

        return res;
    }
}
